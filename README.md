## üöÄ Getting Started

1. **Clone the Repository**:
    ```bash
    git clone https://github.com/Asa-Nguyen/Realtime-Chat-App.git
    cd Realtime-Chat-App
    ```

2. **Install Dependencies**:
    ```bash
    npm install
    ```

3. **Start the Server**:
    ```bash
    npm start
    ```

4. **Access the App**:
   Open your browser and navigate to `http://localhost:3000` to access the app.
